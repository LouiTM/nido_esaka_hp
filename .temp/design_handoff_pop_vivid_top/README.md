# Handoff: nido トップページ「ポップ＆ビビッド」リデザイン

## Overview
nido（大阪・江坂のセルフフォトスタジオ）公式サイトのトップページを、現行の淡いパステル調から、子供向けにより明るくポップでビビッドなトーンへ刷新したデザイン。対象は既存リポジトリ `nido_esaka_hp`（React + TypeScript, Vite想定）のトップページ全セクション（Navbar〜Footer）。

## About the Design Files
同梱の `nido-pop-vivid.html` は **HTMLで作成したデザインリファレンス**（プロトタイプ）です。見た目・挙動の意図を示すものであり、そのままコードとして組み込むものではありません。実装タスクは、このHTMLデザインを、対象リポジトリの既存環境（React 18 + TypeScript + 既存のコンポーネント構成・ビルド設定）にあわせて **再実装** することです。既存の `src/components/*.tsx` の構造・命名パターンを踏襲してください。

## Fidelity
**High-fidelity（hifi）**。色・タイポグラフィ・余白・装飾・インタラクションはこのまま正とし、ピクセルパーフェクトに近い形で再現してください。

## デザインの方向性（サマリー）
- 現行の彩度低めパステル4色（青・コーラル・黄・ピンク）を、そのままビビッド化：青 `#2BA6E8`、コーラル/オレンジ `#FF5A3C`、黄 `#FFC61A`、ピンク `#FF7BAC`
- 全要素に **3px前後の黒(`#1B2224`)インク縁取り** ＋ **ハードシャドウ（ぼかし無し、色付きのオフセットシャドウ）** を使い、ステッカー/おもちゃ的な質感を出す
- カード・バッジ類に軽い `rotate()` の傾きをつけ、静的すぎない配置に
- 各セクション区切りに「スカラップ（半円の連続）」のSVG波形ディバイダー
- ヒーロー・各セクションにガーランド旗、風船、星、丸/角コンフェッティのSVG装飾（お誕生日会のような賑やかさ）
- 見出し英字: **Baloo 2**（丸ゴシック系ポップ体）、和文見出し: **M PLUS Rounded 1c (900)**、本文: **Zen Kaku Gothic New**
- ギャラリー写真は白フチ＋カラー影＋ランダムな傾きで「プリント写真を無造作に並べた」風

## Interactions & Behavior
- 全ボタン・リンク: ホバー時に `transform: translateY(2〜3px)` ＋ ハードシャドウのオフセットを縮小する「押し込み」アニメーション（duration 0.15s）
- Conceptカード: ホバーで回転を0degに戻し、-4px上に浮く（transition 0.2s）
- ギャラリーサムネイル: ホバーで回転リセット＋1.05倍拡大（cubic-bezier(.34,1.56,.64,1), 0.3s）、z-index:2に
- ギャラリークリック → ライトボックスをモーダル表示、背景クリックまたは✕ボタンで閉じる
- 装飾SVG（雲・風船）: CSSアニメーションで常時ゆらゆら/ぷかぷか動く（`nido-drift` 18-22s、`nido-bob` 7-9s、`nido-sway` 10s）。`prefers-reduced-motion: reduce` で停止
- ナビのアンカーリンクはスムーズスクロール（`scroll-behavior: smooth`）

## State Management
- ギャラリーの選択中インデックス（ライトボックス表示用）: ローカル状態で保持、nullで非表示、クリックでインデックスセット
- ヒーロー背景テーマ（3色）: 表示切替用の状態（現在の選択値は「サニーイエロー」）。実装時は例えばテーマ選択のenum stateやCSSカスタムプロパティで良い
- コンフェッティ表示ON/OFF、ギャラリー写真の傾きON/OFFの2つの表示トグル（今回はデザイン確認用のフラグ。本番実装では固定表示でよく、トグルUI自体は不要）

- リンク色: 通常 `#1B7FB8`、ホバー `#E03A1D`
- 本文グレー: `#3A4245` / `#262E31` / 注記 `#6B7478`

### Typography
- 見出し（和文）: `M PLUS Rounded 1c`, weight 900（一部800）
- 見出し（英字ラベル）: `Baloo 2`, weight 600〜800
- 本文: `Zen Kaku Gothic New`, weight 400/500/700
- H1: `clamp(38px, 4.6vw, 64px)` / line-height 1.32
- H2（セクション見出し）: 42px / 900
- 本文: 14〜19px、line-height 1.9〜2.1

### Spacing / Radius / Shadow
- セクション縦パディング: 上72〜80px、下96〜120px
- カード角丸: 32〜36px（ボタン・バッジは pill = 9999px）
- 縁取り: 主要カード/ボタン 3px、チップ/ラベル 2px、常に `#1B2224`
- ハードシャドウ: ぼかし無し・オフセットのみ。例 `8px 8px 0 #FF5A3C`、ボタンは `0 5px 0 #1B2224`（縦方向のみ、押し込み表現用）
- セクション区切り: 高さ40pxのスカラップ（半円連続）SVG、次セクションの背景色で塗る

## Assets
- ロゴマーク: 5枚花びらの円形SVG（インライン、コード内に直接記述。色違いを合図として使用）
- 装飾SVG（雲、風船、星、旗、コンフェッティ、波形アンダーライン、スカラップ区切り）: すべてインラインSVGとしてHTML内に記述済み。アイコン類（LINE, Instagram, 場所ピン, カメラ, 人物, 矢印サイクル）はfeather-icons相当のシンプルな線画。
- ギャラリー写真: 現状プレースホルダー1枚（`nido-about.png`）を使い回し。本番は実際の撮影写真に差し替えが必要
- フォント: Google Fonts経由で読み込み（Baloo 2 / M PLUS Rounded 1c / Zen Kaku Gothic New）
